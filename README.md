# 竞赛小车建模计算器

一个静态 Web 版竞赛小车物理建模计算器。项目基于原 Python 桌面程序的计算逻辑迁移而来，用浏览器完成参数输入、碰撞计算、胜负判断和数学公式展示。

## 功能

- 计算弹簧储能、初速度、平直段速度、坡道段速度、到达时间、碰撞位置、碰后速度和停止距离。
- 判断目标是否达成：我方留在平台，对手被推出平台。
- 支持搜索最小成功后拉距离 `x1`，扫描范围为 `0.01 m ~ 0.20 m`，步长 `0.001 m`。
- 支持同步对手小车参数。
- 支持自动计算坡道长度：`L = H / sin(alpha)`。
- 用 MathJax 渲染 LaTeX 数学公式，每一步包含公式、代入值和结果。
- 支持桌面和手机浏览器，手机端提供底部固定操作栏。
- 无后端、无数据库、无账号密钥，适合 GitHub Pages 静态托管。

## 数学模型概览

主要步骤包括：

1. 弹簧储能：`E_s = 1/2 kx^2`
2. 初速度：`v_0 = x sqrt(eta k / m)`
3. 平直段速度：`v_D^2 = v_0^2 - 2 mu g D`
4. 坡道段速度：`v_P^2 = v_D^2 - 2gL(sin alpha + mu cos alpha)`
5. 到达时间：`t_arr = t_D + t_L`
6. 平台相遇位置：根据先到车的领先距离计算碰撞点 `x_c`
7. 碰撞后速度：使用一维恢复系数碰撞公式
8. 停止距离：`s_stop = v'^2 / (2 mu_p g)`

最终判断：

```text
我方停止距离 < 我方可用平台距离
并且
对手停止距离 > 对手可用平台距离
```

满足时判定成功。
